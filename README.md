# Game Server Manager

**Game Server Manager** は、Windows向けのシンプルでモダンなマルチゲームサーバー管理アプリケーションです。
Palworld、Minecraft、ARKなどの様々なゲームサーバー（exe, bat, cmd等）をひとつのUIで一括管理し、稼働状況を監視します。さらに、サーバーの起動、停止、クラッシュなどのイベントを Discord の Webhook 経由で指定したチャンネルに自動通知する機能を備えています。

## 📥 ダウンロードとインストール

最新版のアプリは **[Releases ページ](https://github.com/LEGOLANNDO/GameServerManager/releases)** からダウンロードできます。

- **Setup.exe (インストーラー版)**: `GameServerManager-vX.X.X-Setup.exe` をダウンロードして実行するだけで、簡単にインストールできます。スタートメニューやデスクトップへのショートカット作成も可能です。
- **Portable.zip (ポータブル版)**: インストールせずに使いたい方向けです。ZIPファイルを解凍し、中にある `GameServerManager.exe` を実行してください。

## ✨ 主な機能

- 🖥️ **モダンなUI (Windows 11 Fluent Design)**: 美しく直感的なダッシュボードで複数のサーバーの稼働状況（CPU / メモリ使用量など）を一目で確認できます。
- 🔔 **Discord 通知の統合**: サーバーの起動、意図的な停止、不意のクラッシュ時にDiscordへリッチなメッセージ（Embed形式）を自動送信します。
- 📅 **スケジュール機能**: 「〇月〇日の〇時〇分に再起動（または停止）」といったスケジュールをアプリ上で予約実行できます。
- 🔐 **セキュアな設計**: DiscordのWebhook URLなどの機密情報は、WindowsのDPAPIを利用して暗号化され、安全に保存されます。
- 📥 **システムトレイ常駐機能**: ウィンドウを閉じてもバックグラウンドで監視を継続します。

## 🚀 使い方

1. アプリを起動し、左側のメニューから **設定 (⚙️)** 画面を開きます。
2. アプリ全体の通知を送るための「グローバル Webhook URL」を登録します。（サーバーごとの個別Webhook設定も可能です）
3. 右上の「サーバー追加」から、管理したいゲームサーバーの実行ファイル（`.exe` や `.bat`）のパスを登録します。
4. **ダッシュボード (🏠)** に戻ると、登録したサーバーがカードとして表示されます。
5. アクションボタン（起動・停止・再起動・スケジュール）でサーバーを操作します。

※ 起動通知について：「起動しました」という通知後、実際のサーバープロセスが完全に立ち上がってプレイヤーが接続可能になるまでには、サーバーの重さによって時差が生じる場合があります。

## 開発とビルドについて

このプロジェクトは C# (.NET 8 WPF) を使用して構築されています。UI には [WPF-UI](https://github.com/lepoco/wpfui) を採用しています。
ビルドやリリース作成は、リポジトリに含まれる PowerShell スクリプト (`build_release.ps1`) と Inno Setup (`installer.iss`) を使用して自動化されています。
